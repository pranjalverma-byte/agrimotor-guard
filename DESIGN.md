# Design Brief: IoT Motor Monitoring Dashboard

**Purpose**: Agricultural precision farming dashboard for real-time DC motor control and temperature monitoring with predictive insights.

**Tone**: Refined minimalism, technical clarity, calm authority — Apple-inspired professional dashboard.

**Color Palette**:
| Token | OKLCH | Purpose |
|-------|-------|---------|
| Primary | 0.48 0.12 205 | Deep teal — trust, technical control |
| Secondary | 0.75 0.08 160 | Soft sage — health, running status |
| Accent | 0.70 0.15 70 | Warm amber — highlights, active states |
| Destructive | 0.55 0.22 25 | Red — alerts, overheating (sparing use) |
| Neutral | 0.995/0.12/0 | Pure white/slate — minimal, clean |

**Typography**: Plus Jakarta Sans (display), General Sans (body), JetBrains Mono (data).

**Elevation & Depth**: Soft, layered shadows—`shadow-card` (1px subtle), `shadow-subtle` (2px), `shadow-elevated` (10px depth).

**Structural Zones**: Header with teal nav bar + logo, main content grid (bg-background), card-based sections (bg-card elevated), footer/status bar muted.

**Spacing**: 8px grid, 16px padding cards, 24px section gaps. Dense information layout with breathing room.

**Motion**: 300ms smooth transitions, motor-pulse animation (2s loop), temp-rise entrance (0.4s). No jarring effects.

**Patterns**: Status indicators (green/red), real-time updates, smooth temperature curves, toggle-switch motor control, alert banners at top.

**Signature Detail**: Minimal rounded corners (10px), refined card layering, teal micro-interactions on hover/focus.
